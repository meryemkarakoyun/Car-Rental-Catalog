# 🚗 CarHub

A modern car rental catalog application built with **React**, **TypeScript**, and **TailwindCSS**. Users can browse vehicles by make, model, and year, view detailed car specifications with multi-angle images, and explore daily rental prices — all powered by the **OpenDataSoft API** and **Imagin.studio** car images.

## 🚀 Features

- 🏎️ **Hero Section** – Animated landing section with a call-to-action to explore cars.
- 🔍 **Make & Model Search** – Filter cars by brand (React Select dropdown) and model (text input).
- 📅 **Year Filter** – Narrow down results by production year.
- 🃏 **Car Cards** – Display make, model, daily price (₺/day), image, transmission, drive type, and year.
- 📋 **Car Detail Modal** – Full detail view with multi-angle car images and formatted specifications.
- 📄 **Pagination** – Browse through results with page navigation (12 cars per page).
- 🔗 **URL-Based Filtering** – Search state synced with URL query parameters (`make`, `model`, `year`, `page`).
- ⚡ **Motion Animations** – Smooth entrance animations on hero and card elements.
- 🌙 **Dark Glass UI** – Modern glassmorphism design with gradient accents.
- 📱 **Fully Responsive** – Optimized layout for all screen sizes.
- ⏳ **Loading & Error States** – Dedicated loader and error components for API requests.
